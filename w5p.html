威科夫(Wyckoff) 投资脑图

一、核心思想 🌍

威科夫方法把市场看作由“供给”与“需求”之间的力量对抗构成的生态系统。价格是这一对抗的实时反映；成交量是力量强度的证据；时间告诉你力量是否被耗尽或在蓄势。投资者的核心任务是通过观察价格、成交量与结构来识别主力（Composite Man）的意图，从而在恰当的阶段做出**纪律性**的交易决策。

**研究提示：** 思考主力如何通过“假突破 + 吸筹/派发”改变市场参与者的行为？用一周/一月级别的图表进行比对。

二、市场四阶段结构（含 Phase A–E） 🔄

把长期区间拆解为更易识别的子阶段，并列举典型事件与判断要点。

积累 — Phase A–E（🟩）

点位：SC → AR → ST → Spring → Test → LPS

Phase A
卖压衰竭，出现 Selling Climax（SC）并伴随自动反弹（AR）。
Phase B
宽幅震荡，主力隐蔽建仓，典型为高位缩量下移与多次测试。
Phase C
Spring（假跌）出现，常引发短期恐慌性抛售，随后被迅速吸纳。
Phase D
LPS（最后支撑点）出现，价格偏离区间下缘并成功回升，趋势准备就绪。
Phase E
Mark-up 启动，趋势性上涨开始。
研究方向：使用 3–6 个月的周线区间，标注 SC 与 Spring，观察后续 1–3 个月的量能变化。
上涨 — Markup（🟦）

特征：高低点抬升、回调缩量、放量突破

识别要点
连续 Higher High / Higher Low；回撤处成交量较低意味着买盘稳定。
操作提示
在回踩 LPS 或重要支撑处分批加仓，止损下移至结构下方。
研究方向：挑选 5-10 只处于 Mark-up 的个股，记录回撤幅度与平均回撤持续时间。
派发 — Distribution（🟥）

点位：BC → AR → ST → UT → UTAD

识别要点
顶部震荡伴随高量但价格无力创新高；出现 UT（向上假突破）诱发散户买入。
警示信号
高位连续放量但价格横盘或回落，注意分批离场。
研究方向：在历史顶部区间查找 UTAD 案例，分析派发周期与平均回撤。
下跌 — Markdown（⬛）

特征：低点下移、放量下跌、支撑失守

判断要点
若跌破多重支撑且伴随放量，下跌趋势确认；短期反弹通常为减仓窗口。
注意事项
观察是否出现新的吸筹迹象以准备下一轮布局。
研究方向：整合多周期（周/月）数据识别下跌周期中早期的吸筹信号。

三、关键要素 📊

价格 / 成交量 / 时间 三元素如何协同工作以形成可靠信号。在威科夫框架中，单一指标不可靠。必须结合：**价位结构**（高低点序列）、**成交量配合**（放量/缩量的相对变化）、以及**时间维度**（震荡持续时间的意义）。

价格结构
高低点的序列决定趋势方向：Higher High / Higher Low 或 Lower Low / Lower High。
成交量
成交量在突破或测试中的作用尤为关键：**放量确认方向**，**缩量提示回撤**为健康调整。
时间
震荡越久，突破后的趋势通常越强；注意**跨周期验证**（周线优先）。
研究提示：用量价热力图或成交量分布图（VPOC）辅助分析吸筹/派发的强度。

四、操作框架 🧩

从信号到执行的标准化流程（便于复制与回测）。

入场（Entry）

条件
处于积累后期（Phase D）或 Mark-up 初期；LPS 或放量突破作为入口。
执行
首笔保守建仓（示例：30%），等待回踩并确认后再分批加仓。

加仓（Pyramiding）

在趋势明确、每次回调**缩量**时分批加仓；每次加仓同时下移止损至上一个结构下方。

止损与仓位管理

止损位放在结构拐点下方（例如 Spring、LPS）。**单笔风险控制 ≤ 2% 账户**，整体仓位根据相关性与波动率调整。

出场（Exit）

出现**派发信号**（UT/UTAD、高位滞涨）时分批出场；目标区间执行分批获利离场，若出现明确反转则全部撤出。

研究方向：设计规则化的入场/止损/出场条件用于回测，并记录每笔交易的入场理由与失误点。

五、图表识别与实战 📈

实战标注规范与多周期协调方法：

  • 区间识别：在**周线**上标注长期区间；在日线中标出短期事件（SC、Spring、LPS、UT 等）。
  • 事件标注：为每次关键事件写短注：时间 / 成交量变化 / 当时市场结构。
  • 量价结合：使用逐笔或分钟数据检验大单吸筹行为（如可能）。
  • 尺度协同：以**周线把握方向**，以日线微调入场与止损。
研究提示：为每次交易在图表上留下注释（入场理由、止损位置、目标、情绪记录），作为复盘素材。

六、心理与纪律 🧘

把纪律写进系统，减少情绪决策：

自我认知
记录交易时的心理状态与触发点（恐惧/贪婪/怀疑），并在复盘时分析其影响。
执行纪律
写明入场与止损规则；若短期内偏离规则，记录原因并调整策略而非情绪处理。
复盘
定期统计胜率、盈亏比、最大回撤与心理偏差，形成改进清单。
研究方向：建立一套交易日志模板，长期跟踪行为偏差并评估改进效果。

七、工具与流程 🧭

支持研究与回测的工具链与流程化建议：

  • **数据：**Akshare / Tushare / Wind（依据权限），注意数据清洗与复权一致性。
  • **可视化：**TradingView（标注）、matplotlib/plotly（回测可视化）。
  • **回测：**Backtrader / zipline / 自研框架；回测需覆盖多种市况与手续费/滑点假设。
  • **流程：**信号判定 → 写入交易单 → 下单执行 → 交易日志 → **周/月复盘**。
研究提示：为回测项目记录假设（滑点、手续费、数据频率），以便结果复现。

📘 实战要点总结

提炼为可执行的研究/交易清单：

  1. 以**周线**为主框架，**日线**为入场细化。
  2. 永远同时观察**价、量、时**三要素。
  3. 使用**结构点**（Spring / LPS / UT）作为明确的止损或出场参考。
  4. 将每次交易写成一句话：**入场理由** + **止损点** + **目标区间** + **心理状态**。
  5. 定期复盘并用**量化指标**评估策略稳定性。
